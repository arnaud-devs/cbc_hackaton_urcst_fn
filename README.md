## 🛠️ Tech Stack

- **Frontend Framework**: [React](https://reactjs.org/) with [TypeScript](https://www.typescriptlang.org/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **UI Components**: [shadcn/ui](https://ui.shadcn.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Routing**: [React Router](https://reactrouter.com/)
- **State Management**: [Redux](https://redux.js.org/)

## 📂 Project Structure

```
HUZACARE_FN/
│
├── public/               # Static files (favicon, index.html, etc.)
│
├── src/
│   ├── assets/           # Static assets like images, fonts, logos
│   ├── components/       # Reusable UI components (buttons, inputs, etc..)
│   ├── context/          # Context API files
│   ├── layouts/          # Reusable layout wrappers (AuthLayout, MainLayout)
│   ├── pages/            # Top-level routes (only if not using App Router)
│   ├── routes/           # Route definitions or configurations
│   ├── store/            # Central Redux store setup, slices.
│   ├── hooks/            # Custom React hooks
│   ├── lib/              # Reusable logic, config, and utility helpers
│   ├── utils/            # Utility functions/helpers
│   ├── types/            # TypeScript types/interfaces
│   ├── App.tsx           # Root component
│   ├── main.tsx          # Entry point
│   └── index.css         # Global CSS
│
├── .env.examples         # Environment variables Examples

```

## 🚀 Getting Started

Follow these steps to set up the project locally:

```bash
# Clone the repository
git clone https://github.com/arnaud-devs/huzacare_fn.git

# Navigate to the project directory
cd huzacare_fn

# Install dependencies
npm install

# Start the development server
npm run dev
```
